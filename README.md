## 🖋️ Sobre a Apresentação

Este repositório contém o material utilizado na minha palestra para os alunos da *FMU. O foco central foi preparar os estudantes para os desafios do **mercado de trabalho*, ensinando como o Git e o GitHub são utilizados em grandes empresas para organizar o desenvolvimento e colaborar em equipe.

Durante o evento, guiei o público através da construção de um projeto *Front-End*, demonstrando que saber programar é apenas metade do caminho; a outra metade é saber gerenciar o seu código profissionalmente.

---

### O que ensinei na prática:
* *Workflow Profissional:* Como organizar as pastas do projeto para que outros desenvolvedores entendam o seu código.
* *Histórico de Evolução:* O uso estratégico de commits para documentar cada melhoria na interface e na lógica do site.
* *Visibilidade:* Como transformar um projeto acadêmico em um portfólio profissional vivo no GitHub.

---

## 🧠 Conteúdo Programático (Soft & Hard Skills)

A palestra não foi apenas técnica, mas também estratégica para a carreira:

1.  *Versionamento Estratégico:* Por que o mercado não aceita mais arquivos como index_final_v2.html.
2.  *Trabalho em Equipe:* Como o GitHub permite que várias pessoas trabalhem no mesmo código sem conflitos.
3.  *Hospedagem e Deploy:* Transformando o código em um link acessível para qualquer recrutador.

---

## 🛠️ Ferramentas Demonstradas

* *Git:* Controle de versão local e comandos de terminal.
* *GitHub:* Hospedagem, documentação (README) e networking técnico.
* *VS Code:* O ambiente de desenvolvimento padrão do mercado.
